# MemoryMatchWear
![Android CI](https://img.shields.io/badge/Android-API26+-green)
![Compose](https://img.shields.io/badge/Jetpack%20Compose-MD3-blue)

Aplicación Android.
Este proyecto implementa el juego MemoryMatchWear directamente en un reloj Wear OS usando Compose for Wear OS, arquitectura MVVM limpia, principio de responsabilidad única y separación de capas.
